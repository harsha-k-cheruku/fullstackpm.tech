## Emergency Troubleshooting

**"I'm not getting interviews!"**
- Customize resumes per application (no generic)
- Add GitHub projects with code samples
- Request referrals from network (refer via LinkedIn connection)
- Target smaller companies first (warm up interviews)
- Check: email going to spam? Use email from recruiter

**"I keep failing coding interviews"**
- Slow down: take 10 mins to think before coding
- Talk through your approach
- Test with examples before submitting
- Review failed problems for patterns
- Practice 10 more similar problems

**"I freeze during interviews"**
- Practice out loud (hearing yourself helps!)
- Do more mock interviews
- Accept some nervousness (normal and okay)
- Remember: they WANT you to succeed (they're hiring!)

**"System design feels impossible"**
- You don't need to be perfect
- Interviewers care about: communication, trade-off thinking, scalability awareness
- Practice sketching (boxes and arrows!)
- Ask clarifying questions ("How many users? What scale?")
